# Time Zone Switcher v0.1.0

Initial open-source release.

## Highlights

- macOS 26+ menu bar app.
- Search and select any IANA time zone.
- Localized time zone display names with IANA identifiers kept visible.
- Toggle on to record the current system time zone and switch to the selected target.
- Toggle off to restore the recorded original time zone.
- Chinese and English README content.

## Installation

Download `Time-Zone-Switcher-v0.1.0-macOS26-arm64.zip`, unzip it, and move `Time Zone Switcher.app` to `/Applications`.

This build is not notarized. On first launch, macOS may require opening it from **System Settings -> Privacy & Security -> Open Anyway**, or by Control-clicking the app and choosing **Open**.
