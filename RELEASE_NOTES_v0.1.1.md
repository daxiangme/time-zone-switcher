# Time Zone Switcher v0.1.1

Maintenance release focused on the menu bar popover layout and project presentation.

## Changes

- Fixed control panel spacing so the time zone list and footer actions sit correctly inside the popover.
- Updated README screenshots with real app interface captures.
- Improved open-source documentation, contribution templates, and issue templates.
- Corrected the MIT license copyright holder.

## Installation

Download `Time-Zone-Switcher-v0.1.1-macOS26-arm64.zip`, unzip it, and move `Time Zone Switcher.app` to `/Applications`.

This build is not notarized. On first launch, macOS may require opening it from **System Settings -> Privacy & Security -> Open Anyway**, or by Control-clicking the app and choosing **Open**.
